Settings

Settings walkthrough
Video coming soon

The Settings tab shows connection info and device configuration.

Info summary

  • Web GUI — browser URL for this device
  • Hotspot login — captive portal entry (http://hivecard/login)
  • Hotspot IP — use this IP for QGroundControl UDP links
  • WiFi name — current hotspot SSID
  • Account — logged-in username
  • PX4 SIH — whether PX4 is installed

WiFi Hotspot

Change the network name (SSID) clients see when scanning for WiFi. Saving may briefly disconnect the hotspot — reconnect using the new name.

Change Login

Update username and/or password. You must enter your current password. Password must be at least 4 characters; username at least 3.

Build SITL Binaries

ArduPilot vehicles must be compiled before they appear in the New panel. Each row shows a vehicle that needs building — tap Build and wait for completion. The page reloads when done.

Sign Out

Ends your session. You will need to sign in again, especially after a device reboot.

Help & Instructions in Settings opens on-device docs at /help, or read online at jamesuasop.github.io/hivecard.
