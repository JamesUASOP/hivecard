The Logs tab shows recent system output. It refreshes automatically while you are on that panel.

Common issues

Vehicle not in New panel
Build the ArduPilot SITL binary in Settings → Build SITL Binaries, then reload the page.
PX4 option disabled
PX4 SIH is not installed on this device. Check Settings → PX4 SIH for status.
Cannot start another aircraft
All three slots (AC01–AC03) may be in use. Stop one aircraft or use Stop All.
QGC shows no data
Confirm you are on HiveCard WiFi, GCS status is Ready, and you are using the correct UDP port and Hotspot IP from the fleet card.
WiFi name change failed
Hotspot controls may need to be installed on the device. Contact your administrator if you see a permissions error.
Lost WiFi after SSID change
The hotspot reconnects with the new name. Join the updated network and open http://hivecard/login again.
