Connect QGroundControl

{{ video_slot('connect_qgc', videos) }}

Each running aircraft exposes a UDP port on the HiveCard hotspot. Use the address shown on the aircraft card under UDP.

Connection steps

  1. Connect your phone or tablet to the HiveCard WiFi network.
  2. Start an aircraft on the Dashboard and wait until GCS shows Ready.
  3. In QGroundControl, add a UDP communication link.
  4. Enter the HiveCard Hotspot IP (from Settings, often 10.42.0.1) and the port from the aircraft card.
  5. Connect and verify telemetry. Match the SysID on the card if running multiple aircraft.

Multiple aircraft

Run separate QGC links (or instances) for each aircraft, each on a different UDP port. Always confirm the SysID matches the slot you intend to control.
