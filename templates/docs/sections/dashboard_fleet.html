Dashboard & Fleet

{{ video_slot('dashboard_fleet', videos) }}

The Dashboard shows every simulated aircraft currently running on HiveCard. You can run up to 3 aircraft at once (AC01, AC02, AC03).

Toolbar

  • Connected | Errors — summary of fleet health
  • N/3 Aircraft — how many slots are in use
  • Stop All — stops every running simulator
  • + Add — opens the New panel to start another aircraft

Aircraft card

Each card shows:

  • Status — Running, Starting, or Stopped
  • Stack — ArduPilot or PX4
  • Firmware and Airframe
  • Connection — GCS readiness, UDP address, and SysID
  • Location — spawn name and coordinates
{% if not (docs_brief|default(false)) %}
Tip: Wait until GCS shows Ready before connecting QGroundControl. While status is Starting, the simulator is still booting.

Use Stop on a card to shut down one aircraft, or Stop All to clear the entire fleet.
