Getting Started

{{ video_slot('getting_started', videos) }}

HiveCard is a mobile simulation tool that runs software-in-the-loop (SITL) aircraft on a Raspberry Pi and broadcasts them over WiFi so you can connect with QGroundControl or another GCS.

Quick start

  1. On your phone, tablet, or laptop, open WiFi settings and join the HiveCard network. The default network name is HiveCard (you can change it later in Settings).
  2. Open a browser and go to http://hivecard/login or http://10.42.0.1:8080.
  3. Sign in with your HiveCard username and password. The factory default is hivecard / hivecard until you change it in Settings.
  4. Use the bottom navigation to move between Dashboard, New, Logs, Locations, Settings, and Help.
