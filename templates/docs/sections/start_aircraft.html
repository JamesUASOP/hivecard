Start Aircraft

{{ video_slot('start_aircraft', videos) }}

Open the New tab to launch a simulator. HiveCard supports ArduPilot and PX4 stacks on the same device (mixed fleet).

Steps

  1. Choose Autopilot Stack — ArduPilot or PX4 (PX4 requires SIH to be installed on the device).
  2. Select vehicle type and frame (ArduPilot) or PX4 airframe (SIH model).
  3. Pick a Starting Location from your favorites or full list.
  4. Tap Start Aircraft. You are returned to the Dashboard while the sim boots.
{% if not (docs_brief|default(false)) %}

ArduPilot

  • Choose a vehicle such as Copter, Plane, or Rover, then pick a frame from the icon grid.
  • The SITL binary must be built first — go to Settings → Build SITL Binaries if your vehicle is missing from the list.
  • Location heading is applied when spawning ArduPilot aircraft.

PX4

  • Select a PX4 SIH airframe: Quadcopter, Hexacopter, Standard VTOL, or Rover.
  • PX4 uses latitude, longitude, and MSL altitude only — heading is ignored.
Note: If all three slots are full, the + Add button is disabled until you stop an aircraft.
