Locations

{{ video_slot('locations', videos) }}

Locations define where simulated aircraft spawn. Open the Locations tab to manage them.

List view

  • Search by name using the search box.
  • Filter by My Locations, Favorites, or All.
  • Tap + Add New Location to create a custom spot.
  • Use Edit or Del on user-created locations. Built-in ArduPilot imports cannot be deleted.

Add or edit a location

  1. Enter a Name (required).
  2. Set Latitude, Longitude, Altitude (m), and Heading.
  3. Optionally set category, description, and mark as favorite.
  4. Tap Save Location or Update Location.
{% if not (docs_brief|default(false)) %}
Heading: Applies to ArduPilot spawns only. PX4 SIH uses latitude, longitude, and MSL altitude.

When starting an aircraft, open the New tab and pick a location from the Starting Location list. Tap Need a new spot? to jump to Locations.
